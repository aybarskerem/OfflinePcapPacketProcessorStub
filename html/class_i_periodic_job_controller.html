Detailed Description

thread-safe interface class which represents a controller for IPeriodicJob.

This interface contains a container holding each IPeriodicJob it starts and methods to add new jobs, remove added ones given a specific IPeriodicJob, remove an arbitary one or change the period of a specified job.
