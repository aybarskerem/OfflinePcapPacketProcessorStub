void processPackets ( )

A free function which is designed to run continuously in a thread and process newly arrived pcap packets.

This function calls uses Common::kMaxNumberOfPacketsToProcess constant to determine how many packets to process. The function prototype can be changed to accept parameter to determine this number for each different call instead.
