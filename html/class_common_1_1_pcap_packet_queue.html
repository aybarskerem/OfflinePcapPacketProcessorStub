Detailed Description

A thread-safe queue-like FIFO data structure & Singleton class to hold and provide Pcap packet data (PcapPacket) via queue-like methods.

Whenever this class's methods are requried; firstly getInstance method of this class should be called to get the one and only instance of this class and the methods should be called on this instance.

This class has an internal container holding PcapPacket class instances where anyone can push or pop&read those.
