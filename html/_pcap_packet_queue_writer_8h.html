Detailed Description

This file contains free functions simulating pushing packets to PcapPacketQueue.

In a real-world application; we do not need the functions defined here. However, for this application, we need to receive some pcap packets from external world. To simulate an external world; these functions are used so that the PcapPacketQueue has packets waiting to be consumed.
