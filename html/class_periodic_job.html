Detailed Description

A thread-safe class which implements IPeriodicJob.

I could have virtually inherited (i.e. virtual public IPeriodicJob) from IPeriodicJob in case it is possible that we'd have multiple child classes inheriting from PeriodicJob and then another class inheriting from these children which want access to IPeriodicJob member variables (causing two or more instance of the same member). However in our code no diamond inheritance concerns exists; hence no need for virtual inheritance.
